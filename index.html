Tableau Dashboard for Covid

  • Set of queries were run is MSSQL to get the desired data.
  • This dashboard was made to monitor death and infection ratio happened due to COVID.
  • It can be used for anyone who wants to know which continents population is highly infected can be easily figured it through the map.
  • It can also be used to check the death percentage which at that time was 2% and can be used in future on daily basis to keep track of the death percentage overall and also based on the location.
  • To make it an attractive dashboard and easy for visualisation some table and charts were used such as, text tables, horizontal bar chart, line chart and symbol map.
  • Using git commands, pushed the project contents from local to GitHub Tableau Covid Dashboard repository and uploaded the dashboard to Tableau Public Online.
